# Proyek-Akhir-Pemrosesan-Teks---Penyajian-Informasi-Kesehatan-Sebagai-Edukasi-Publik
Proyek ini melakukan web scraping data artikel dari Nakama Aquatics, kemudian menerapkan TF-IDF untuk analisis teks dan K-Means serta Hierarchical Clustering untuk pengelompokan artikel berdasarkan kategori. Visualisasi dilakukan menggunakan Word Cloud dan evaluasi clustering dengan Metode Elbow serta dendrogram.
## Fitur Utama:
- **Web Scraping**: Mengambil data artikel dari **Nakama Aquatics** menggunakan **Python**.
- **Pre-processing**: Pengolahan teks dengan **stemming**, **remove punctuation**, dan **stopword removal**.
- **Clustering**: Implementasi **K-Means** dan **Hierarchical Clustering** untuk pengelompokan artikel.
- **Evaluasi**: Penggunaan **Metode Elbow** untuk K-Means dan **dendrogram** untuk Hierarchical Clustering.
## Teknologi yang Digunakan:
- Python
- **K-Means** Clustering
- **Hierarchical Clustering** (Ward Method)
- **TF-IDF** (Term Frequency-Inverse Document Frequency)
- **NLTK**, **Gensim** (Word2Vec)
- **WordCloud**, **Plotly**
