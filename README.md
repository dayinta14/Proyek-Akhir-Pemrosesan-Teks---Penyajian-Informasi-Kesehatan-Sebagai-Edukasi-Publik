# Proyek-Akhir-Pemrosesan-Teks---Penyajian-Informasi-Kesehatan-Sebagai-Edukasi-Publik
Proyek ini melakukan web scraping data artikel dari Nakama Aquatics, kemudian menerapkan TF-IDF untuk analisis teks dan K-Means serta Hierarchical Clustering untuk pengelompokan artikel berdasarkan kategori. Visualisasi dilakukan menggunakan Word Cloud dan evaluasi clustering dengan Metode Elbow serta dendrogram.
